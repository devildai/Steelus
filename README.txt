    ├── css/

    └── images/

WHAT EACH FILE DOES

1. index.html
   Main webpage file.

2. assets/css/style.css
   All design, colors, spacing, responsiveness.

3. assets/js/script.js
   Small JavaScript for mobile menu and footer year.

4. assets/images/logo.jpg
   Your company logo image. Replace with a better logo version later if you want.

5. assets/images/hero-steel.jpg
   Main banner image shown on homepage.

6. assets/images/steel-1.jpg to steel-4.jpg
   Gallery images for steel product photos.

HOW TO USE

1. Open the folder.
2. Double-click index.html to test in browser.
3. Replace placeholder images with your real company images.
4. Keep the SAME file names unless you also update the names inside index.html.

WHERE TO PUT YOUR IMAGES

- Put main logo inside: assets/images/logo.jpg
- Put banner image inside: assets/images/hero-steel.jpg
- Put gallery steel images inside:
  assets/images/steel-1.jpg
  assets/images/steel-2.jpg
  assets/images/steel-3.jpg
  assets/images/steel-4.jpg

IF YOU WANT TO ADD MORE GALLERY IMAGES

Copy one of these inside the gallery section in index.html:

<figure class="gallery-item">
  <img src="assets/images/steel-5.jpg" alt="Steel product image 5" />
</figure>

Then place steel-5.jpg inside assets/images/.

IF YOU WANT TO CHANGE COMPANY DETAILS

Open index.html in VS Code or Notepad++ and edit:
- Company name
- Owner name
- Address
- Phone numbers
- Product names
- About text
